# Wavespeed AI - SeedVR2 Upscaler for Forge

![License](https://img.shields.io/badge/License-MIT-green.svg) ![Platform](https://img.shields.io/badge/Platform-SD%20WebUI%20Forge-blue.svg) ![Cost](https://img.shields.io/badge/Cost-%240.01%2Fupscale-yellow.svg)

A dedicated extension for **Stable Diffusion WebUI Forge** that integrates the Wavespeed AI (https://wavespeed.ai) **SeedVR2 Upscaler**. This tool allows you to seamlessly upscale your generated images to **2k, 4k, or 8k** resolution directly within the Forge interface for just **$0.01 USD per upscale**.

It features a streamlined workflow with "Grab Last Generated" functionality and a built-in aspect ratio corrector (center crop) to fix non-standard resolutions (e.g., trimming 1920x1088 to 1920x1080) prior to upscaling.

* **Secure Config**: Store your API key permanently in the Forge Settings or enter it per session.

## ğŸ“¥ Installation

### Method 1: Install via Extension URL
1. Open **WebUI Forge**.
2. Navigate to the **Extensions** tab -> **Install from URL**.
3. Paste the repository URL into the field:
   https://github.com/sdprompts/forge-wavespeed-seedvr2-upscaler
4. Click **Install**.
5. Go to the **Installed** tab and click **Apply and Restart UI**.

### Method 2: Manual Installation
1. Navigate to your Forge `extensions` folder in your terminal:
   
   cd stable-diffusion-webui-forge/extensions

2. Clone this repository:
   
   git clone https://github.com/sdprompts/forge-wavespeed-seedvr2-upscaler.git

3. Restart the WebUI.

## âš™ï¸ Configuration

### Setting the API Key (Recommended)
You can save your API key globally so you don't need to enter it every time.

1. Open Forge and go to the **Settings** tab.
2. Scroll down the left-hand menu (or use the sidebar) and select **Wavespeed AI**.
3. Paste your key into the **Wavespeed API Key** field.
4. Click **Apply Settings** (usually at the top or top-right in Forge).

*(Alternatively, you can paste your key directly into the "API Key" box on the Wavespeed tab for one-time use.)*

## ğŸ–¥ï¸ Usage Guide

1. **Open the Tab**: Click on the **Wavespeed AI** tab in the main Forge interface.
2. **Select Input Source**:
   * **Local / Upload**: Drag and drop an image, or click **"ğŸ“… Grab Last Generated Image"** to auto-load your latest generation.
   * **URL**: Switch to the "Image URL" tab to upscale a remote image link.
3. **Adjust Settings**:
   * **Target Resolution**: 2k, 4k, or 8k.
   * **Format**: PNG, JPEG, or WEBP.
   * **Pre-Upscale Crop**: (Optional) Check "Enable Center Crop" and set dimensions (default 1920x1080) to trim edges before sending to the API.
4. **Upscale**: Click the **Upscale** button. The cost is **$0.01** per transaction.
5. **View Results**: The processed image will appear in the result window and be saved to your output directory.
