# Wavespeed AI - SeedVR2 Upscaler for Forge

![License](https://img.shields.io/badge/License-MIT-green.svg) ![Platform](https://img.shields.io/badge/Platform-SD%20WebUI%20Forge-blue.svg) ![Cost](https://img.shields.io/badge/Cost-%240.01%2Fupscale-yellow.svg)

A dedicated extension for **Stable Diffusion WebUI Forge** that integrates the Wavespeed AI (https://wavespeed.ai) **SeedVR2 Upscaler**. This tool allows you to seamlessly upscale your generated images to **2k, 4k, or 8k** resolution directly within the Forge interface for just **$0.01 USD per upscale**.

It features a streamlined workflow with "Grab Last Generated" functionality, a built-in aspect ratio corrector (center crop), and a **0.35 Megapixel optimizer** to ensure the highest possible texture quality from the SeedVR2 model.

## ğŸ“¥ Installation

### Method 1: Install via Extension URL
1. Open **WebUI Forge**.
2. Navigate to the **Extensions** tab -> **Install from URL**.
3. Paste the repository URL into the field:
   https://github.com/sdprompts/forge-wavespeed-seedvr2-upscaler
4. Click **Install**.
5. Go to the **Installed** tab and click **Apply and Restart UI**.

### Method 2: Manual Installation
1. Navigate to your Forge `extensions` folder in your terminal:
   
   `cd stable-diffusion-webui-forge/extensions`

2. Clone this repository:
   
   `git clone https://github.com/sdprompts/forge-wavespeed-seedvr2-upscaler.git`

3. Restart the WebUI.

1. **Open the Tab**: Click on the **Wavespeed AI** tab in the main Forge interface.
2. **Select Input Source**:
   * **Local / Upload**: Drag and drop an image, or click **"ğŸ“… Grab Last Generated Image"** to auto-load your latest generation.
   * **URL**: Switch to the "Image URL" tab to upscale a remote image link.
3. **Adjust Pre-Processing**:
   * **Downscale to 0.35MP**: (Recommended) Keep this checked for the best SeedVR2 results. It cleans the image and lets the AI reconstruct details from a clean base.
   * **Pre-Upscale Crop**: (Optional) Check "Enable Center Crop" to trim edges (e.g., 1920x1088 to 1080) before processing.
4. **Select Target**: Choose **2k, 4k, or 8k** and your preferred format.
5. **Upscale**: Click the **Upscale** button ($0.01 per transaction).
6. **View Results**: The processed image will appear in the result window and be saved to your output directory.

## ğŸ› ï¸ Troubleshooting

**"Why should I use the 0.35MP option?"**
> SeedVR2 is a "tiled" diffusion upscaler. If you feed it a very high-res but slightly blurry image, it can sometimes produce "waxy" skin. Downscaling to 0.35MP first forces the model to re-generate those details, resulting in much sharper pores, hair, and textures.

**"Error: Please enter your API Key..."**
> Ensure you have pasted your key in the Settings tab or the text box on the page.

**"Submission Error..."**
> Check your internet connection, verify your API key is valid, and ensure you have sufficient credits on your Wavespeed account.
